##### ⚙️ Настройка ПО String.

**> Это необязательная процедура, она нужна только для вашего удобства, можно легко перейти к стандартным настройкам.**



* **string --save off/on** 

Включает или отключает сохранение отчета после поиска. Важно, отчет сохраняется в виде сырого JSON объекта.



* **string --g1 1,1,1,1 --g2 2,2,2,2 \[New 16.03.2026]**

g1 принимает первый цвет из палитры RGB, g2 принимает второй цвет из RGB палитры, в дальнейшем эти цвета будут использоваться для градиента.



* **string --c1 #E0FFFF --c2 #E0FFFF \[New 16.03.2026]**

с1 принимает HEX цвет из палитры HEX, c2 принимает HEX цвет из палитры HEX



* **string --remove \[New 15.03.2026]**

Сбрасывает все настройки
