## ⚙️ Настройка ПО

> 🧪 **Необязательно** — процедура нужна только для вашего удобства.  
> Можно легко вернуться к стандартным настройкам.

---

### 🎨 Интерфейс и внешний вид

**`string --save off/on`**  
Включает или отключает сохранение отчёта после поиска. Отчёт сохраняется в виде сырого JSON-объекта.

**`string --remove`** `[New 15.03.2026]`  
Сбрасывает все настройки к заводским.

---

### 🌈 Градиент и цвета
`[New 16.03.2026]`

**`string --g1 1,1,1,1 --g2 2,2,2,2`**  
`g1` принимает первый цвет из палитры RGB, `g2` — второй цвет. В дальнейшем эти цвета используются для градиента.

**`string --c1 #E0FFFF --c2 #E0FFFF`**  
`c1` принимает HEX-цвет, `c2` принимает HEX-цвет. Альтернативный способ задания палитры.

## 🔎 Поиск по базам

> 📡 **Важно:** Поиск происходит по платному источнику.  
> Источник наделён реальными базами данных.

---

### Поиск по номеру телефона
**`string --api-phone 79000000000`**  
Принимает аргумент в виде номера телефона. Поиск только по номеру.

---

### Поиск по ФИО
**`string --api-people Пвахин Стринг Питонович`**  
Принимает аргумент в виде ФИО. Поиск только по персональным данным.

---

### Поиск по email
**`string --api-mail pwaexho@string.com`**  
Принимает аргумент в виде почтового адреса. Поиск только по email.

---

### Поиск по VIN-номеру
**`string --api-vin 1FAFP53U7YG164887`**  
Принимает аргумент в виде VIN-номера автомобиля. Поиск только по автоистории.

---

### Поиск по паролю
**`string --api-pass string2026`**  
Принимает аргумент в виде пароля. Поиск только по парольным базам.

---

### Поиск по СНИЛС
**`string --api-snils 541-251-383 52`**  
Принимает аргумент в виде СНИЛС. Поиск только по пенсионным данным.

---

### Поиск по ИНН
**`string --api-inn 37507352790`**  
Принимает аргумент в виде ИНН. Поиск только по налоговым данным.

---

### Поиск по профилю ВКонтакте
**`string --api-vk https://vk.com/string`**  
Принимает аргумент в виде ссылки на профиль. Поиск только по социальным сетям.

---

### Поиск по IP-адресу
**`string --api-ip 8.8.8.8`**  
Принимает аргумент в виде IP-адреса. Поиск только по геолокации и сетям.

## 📃 Управление справкой

**`string --thelp`** (Txt Help)  
Создаёт TXT-файл с подробной справкой по использованию STRING.

**`string --shelp`** (Shell Help)  
Выводит в консоль компактную справку по использованию STRING.

**`string --mdhelp`** (MarkDown Help)  
Создаёт MD-файл с документацией (как этот).
